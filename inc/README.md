# inc/ — Reconstructed Win 1.x headers

This directory contains MASM-style `.inc` reference files reconstructed from
the public Windows 1.0 SDK reference, Petzold's *Programming Windows* (1988),
and the NE export tables.

These files are **not currently included** by the seg*.asm sources (those
sources are byte-exact via raw `db` directives). They serve as:

1. Documentation — browse all known APIs and constants in one place.
2. Future scaffolding — when individual functions are progressively lifted
   to a more semantic form, these declarations provide the symbol space.
3. Clean-room reference — anyone attempting a clean-room reimplementation
   can use these as the public-API boundary they must respect.

## Files

| File         | Contents                                          |
|--------------|---------------------------------------------------|
| windows.inc  | Common types, WM_* messages, VK_* keys, GMEM_*,   |
|              | LMEM_*, WS_*, SW_*, ROP codes, MB_* styles        |
| kernel.inc   | KERNEL.EXE exported procedure prototypes          |
| user.inc     | USER.EXE exported procedure prototypes            |
| gdi.inc      | GDI.EXE exported procedure prototypes             |
| cmacros.inc  | Standard FAR PASCAL prologue/epilogue macros      |
